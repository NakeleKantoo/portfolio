# Portfolio

This repository contains the source code for my personal portfolio website, built using React and Tailwind CSS. The website showcases my projects, skills, and contact information.

## Features

- Project showcase with descriptions and links
- Contact section with links to social media and email

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/NakeleKantoo/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:5173` to view the website.
